Emotional GIF Viewer App

This web application draws from the GIPHY API to populate with gifs that visually represent basic human emotions. The application presents a pre-populated list of emotions to select from but also allows the user to add any other emotions that they might find interesting. This application uses a number of web technologies including HTML, CSS, Javascript, jQuery, and AJAX. Enjoy!

Train Scheduler App

This web application allows the the user to quickly understand the proximity of upcoming trains coming and going from Ballston MU metro station by providing the user with the time of the next arriving train and the number of minutes until its arrival. This application uses a number of web technologies including HTML, CSS, Javascript, jQuery, AJAX, and Moment.Js
